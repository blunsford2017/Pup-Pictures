# Pup Pictures

A simple web page for your daily dose of cute pup pictures! 

## Technologies used:
- HTML 5
- CSS 3
- JavaScript
- jQuery 
- Google Fonts
- Dog API
- Dog Town: Pet Shop Game, Care & Play with Dog APP
- DogWorld 3D: My Puppy APP
- DogHotel APP
- TamaDog! - AR Tamagotchi Game APP
- Netlify
